# CBRC Tutorial Site
This site is written in Quarto. You can find the installation instructions for Quarto [here](https://quarto.org/docs/download/).

To render Quarto `.qmd` files to HTML:
```bash
quarto render
```

To preview the site from your local machine:
```bash
quarto preview
```

To render and deploy website (after all changes are commited and pushed to origin main):
```bash
quarto publish gh-pages
```
